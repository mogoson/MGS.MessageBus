# MGS.MessageBus

## Summary

- Bus for message subscribe and unsubscribe and spread.

## Install

- Unity --> Window --> Package Manager --> "+" --> Add package from git URL...

  ```text
  https://github.com/mogoson/MGS.MessageBus.git?path=/Assets
  ```

## Usage

- Create a Global Instance for IMessageBus.
- Subscribe or unsubscribe message from the Global Instance.
- Spread message by the Global Instance.

## Samples

- Unity --> Window --> Package Manager --> Packages-Mogoson --> Message Bus --> Samples.
